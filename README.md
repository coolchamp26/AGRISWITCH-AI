# AgriSwitch AI 🌾

**AI-Powered Crop Residue Decision Intelligence System for India**

Helps Indian farmers make economically rational decisions about crop residue (stubble) management by comparing burning vs sustainable alternatives with transparent calculations, AI-powered explanations, and government scheme guidance.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm
- Supabase account (for auth & database)
- OpenAI API key (for AI features)

### Setup

1. **Clone and install:**
```bash
cd HACK4GREEN
npm install
```

2. **Configure environment:**
```bash
cp .env.example .env.local
# Edit .env.local with your keys:
# NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
# NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
# OPENAI_API_KEY=your_openai_key
```

3. **Setup Supabase database:**
   - Go to your Supabase dashboard → SQL Editor
   - Run the contents of `supabase/schema.sql`

4. **Run locally:**
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

## 🏗️ Architecture

```
Deterministic Engine (calculations.ts)    ← ALL numbers computed here
         ↓
    API Routes (Next.js)                  ← Server-side LLM calls
         ↓
OpenAI GPT-4o (Explain ONLY)             ← Never generates numbers
```

**Key principle:** Math is deterministic. AI only explains pre-calculated values.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🧮 Deterministic Engine | All calculations in `calculations.ts` — no LLM math |
| 🤖 AI Explanations | GPT-4o explains pre-calculated numbers in EN/HI |
| 📋 Policy RAG | Static JSON dataset of real CRM schemes |
| 📄 PDF Generation | Download subsidy/loan application drafts |
| 🎛️ Scenario Simulator | Sliders for diesel/labor cost changes |
| 📊 Impact Dashboard | Recharts charts for CO₂, costs, benefits |
| 🌐 Multilingual | English + Hindi toggle |
| 🔐 Auth | Supabase email OTP (+ demo mode) |
